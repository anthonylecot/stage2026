Cyber / OSINT

Enquête sur un inconnu avec des outils publics

Objectif

Découvrir ce qu'on peut trouver sur quelqu'un uniquement avec des informations publiques — et comprendre ce que ça dit de ta propre empreinte numérique.

OSINT = Open Source INTelligence. Renseignement à partir de sources ouvertes.

La mission

Tu dois retrouver un maximum d'informations sur une personne à partir d'un seul indice de départ : un pseudonyme.

Pseudonyme de départ : amensch42

Acte 1 — WhatsMyName

Aller sur whatsmyname.app et chercher le pseudonyme.

Cet outil scanne des centaines de plateformes pour voir si le pseudo existe.

  • Sur quelles plateformes le pseudo apparaît-il ?
  • Lesquelles semblent actives ?

Acte 2 — Google Dorks

Les Google Dorks sont des requêtes avancées pour trouver des informations précises.

SyntaxeEffet
"amensch42"Cherche exactement ce texte
site:github.com amensch42Cherche uniquement sur GitHub
"arthur mensch" site:linkedin.comCherche un nom sur LinkedIn
filetype:pdf "arthur mensch"Cherche des PDF contenant ce nom

Acte 3 — GitHub

Explorer le profil GitHub trouvé :

  • Quelle est la bio ? Que révèle-t-elle ?
  • Y a-t-il des dépôts publics ?
  • Y a-t-il une adresse ou localisation mentionnée ?

Acte 4 — Street View

Si une adresse ou un lieu est mentionné, utiliser Google Street View pour le visualiser.

Que peut-on déduire de ce qu'on voit ?

Débrief — Ton empreinte numérique

Après l'enquête : que peut-on trouver sur toi avec les mêmes outils ?

  • Tes comptes sont-ils publics ou privés ?
  • Qu'est-ce que tu veux que les autres voient ?
  • Ton site GitHub Pages est public — qu'y as-tu mis ?
