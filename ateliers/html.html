HTML + CSS + GitHub Pages

Construire et publier ton site personnel

Objectif

À la fin de cet atelier, tu auras publié un site HTML personnel sur GitHub Pages. Ce site sera ton portfolio pour tout le stage — tu y ajouteras une page après chaque atelier.

Étape 1 — Mettre en place VS Code

  1. Ouvrir VS Code
  2. Créer un dossier mon-site sur le bureau
  3. File → Open Folder → sélectionner mon-site
  4. Dans l'explorateur à gauche : clic droit → New File → nommer index.html
VS Code en 30 secondes : explorateur de fichiers à gauche, éditeur au centre, terminal en bas (Ctrl+ù). Pour voir ta page dans Chrome : glisse le fichier index.html dans le navigateur.

Étape 2 — Le squelette HTML

Colle ce code dans ton index.html :

<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <title>Mon site</title>
</head>
<body>

  <h1>Bonjour, je m'appelle Prénom</h1>
  <p>Je suis stagiaire chez SAP en juin 2026.</p>

</body>
</html>
BaliseRôle
<!DOCTYPE html>Dit au navigateur que c'est du HTML5
<head>Informations sur la page (non visibles)
<meta charset="UTF-8">Permet les accents
<title>Texte dans l'onglet du navigateur
<body>Tout ce qui s'affiche dans la page
<h1>Titre principal
<p>Paragraphe

Étape 3 — Les balises essentielles

<!-- Titres -->
<h1>Titre principal</h1>
<h2>Sous-titre</h2>

<!-- Texte -->
<p>Un paragraphe.</p>
<strong>Texte en gras</strong>
<em>Texte en italique</em>

<!-- Liste -->
<ul>
  <li>Élément 1</li>
  <li>Élément 2</li>
</ul>

<!-- Lien -->
<a href="https://sap.com">Visiter SAP</a>

<!-- Image -->
<img src="photo.jpg" alt="Description">

Étape 4 — Ajouter du CSS

Créer un fichier style.css dans le même dossier. Puis lier le CSS dans le <head> de chaque page HTML :

<link rel="stylesheet" href="style.css">

Dans style.css :

body {
  background-color: #f5f5f5;
  color: #333333;
  font-family: Arial, sans-serif;
  max-width: 800px;
  margin: 0 auto;
  padding: 20px;
}

h1 {
  color: #0070ad;
}

a {
  color: #0070ad;
}

Étape 5 — Publier sur GitHub Pages

Dans le terminal VS Code (Ctrl+ù) :

git init
git add .
git commit -m "Premier commit - mon site de stage"
git branch -M main
git remote add origin https://github.com/TON_PSEUDO/mon-site.git
git push -u origin main

Ensuite sur GitHub : Settings → Pages → Deploy from branch → main → Save

Ton site sera accessible à : https://TON_PSEUDO.github.io/mon-site

Pour mettre à jour ton site après chaque atelier :
git add . → git commit -m "Ajout page Scratch" → git push
